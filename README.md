## Overview

![screenshot](/thumbnail.jpg)

This project is a simple FAQ page built as a solution to the devChallenges.io Simple FAQ challenge. It displays frequently asked questions with clear, structured answers using semantic HTML and custom CSS styling.

### What I learned

During this project, I improved my skills in:

- Writing semantic HTML5 markup for better accessibility and SEO
- Using CSS custom properties (variables) for maintainable styling
- Structuring content in a clean, readable format
- Creating responsive layouts with CSS
